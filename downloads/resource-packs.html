Minecraft Resource Packs

What are Resource Packs?

Minecraft resource pack can change the way the game looks, through modifying .png files and .json files.

Piglin to Pigman

Minecraft Bedrock 1.18+ [Java port available]

Turns the 1.16 Zombie Piglins back to pre 1.16 Zombie Pigman.

Better Netherite

Minecraft Bedrock 1.18+ [Java port available]

Gives Netherite a different texture, closer to ruby to make it match the colour palette of the Nether more.

Updated 1.8 textures

Minecraft Java 1.6.1 - 1.8

Backports the latest Minecraft Textures to be used in older versions, mostly for 1.8 PvP.

Pano selector

Minecraft Bedrock 1.18+

Allows you to select a range of different panos from previous/different versions of the game.

Clear chat

Minecraft Bedrock 1.18+

Removes the chat background when playing.

Green screen pack

Minecraft Bedrock 1.18+

Make's certain blocks completely green or blue for your greenscreen needs.

Xmas chests

Minecraft Bedrock 1.18+

Adds in the xmas chests from Minecraft Java edition.

Custom Fog

Minecraft Bedrock 1.18+

Adds in a slider to lets you make fog scalable with your render distance in Liquids and the Nether, or both.
